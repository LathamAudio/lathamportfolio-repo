Unity+Wwise - 3dGameKitWwise

A Wwise implementation of the 3dGame Kit in the Unity Engine

DOWNLOAD HERE

https://drive.google.com/file/d/12sdJIlbeuURle7mnvwCzz62_KizaedJl/view?usp=sharing

Intallation

  1. Download & install the Unity Hub at http://unity.com/

  2. Install Unity 2022.3.59f1

Create 3dGameKitWwise Project 3. Create a new Unity project using the 3d template, call it something like “3dGameKitWwise” and save it in your favorite file location

  1. Once the project is open navigate to the Asset Store found at http://assetstore.unity.com

  2. Search for “3d Game Kit” in the Unity Asset Store. Download and import it into your project, approving any popups (this can take a while).

Open first 3d Game Kit scene 6. Once complete you can find and open the Start scene by going to “Assets\3dGameKit\Scenes" in the Project window

  1. Try playing 3dGameKit to see if will run by pressing Play button, once successful you can close Unity

  2. Download 3dGameKitWwise_WwiseProject.zip here and unzip the contents into the 3dGameKitWwise Unity project folder.

Install Wwise 10. Download & install the Wwise Launcher at http://www.audiokinetic.com/

  1. Install Wwise 2019.1.8 You only need to select the Authoring tool & SDK at this time, no plugins are needed for this project

  2. In the Wwise Launcher go to the Unity tab and choose “Integrate Wwise Into Project…” for the 3dGameKitWwise Unity project

Integrate Wwise Into 3dGameKitWwise Project 13. You may need to update Wwise to integrate it into the project, press “Modify Wwise…”

  1. This will take you back to Wwise install screen with all of the necessary packages and platforms already selected, press “Modify” and again you don’t need to install plugins

  2. Once that has completed successfully, press “Add WWise to 3dGameKitWwise”

  3. Next set the Wwise Project Path by navigating to your Unity project and into the 3dGameKitWwise_WwiseProject folder to select the “3dGameKitWwise_WwiseProject.wproj” file

  4. Press the “Integrate” button to integrate Wwise into the Unity project

Generate Wwise Soundbank 18. Once complete open the “3dGameKitWwise_WwiseProject.wproj” session in Wwise. It may ask to update the project, press “Yes”

  1. Go to the Soundbank layout and generate the soundbank choosing PC or MAC depending on the platform

Play and Test 3dGameKitWwise

  1. Open Unity and the 3dGameKitWwise project

  2. From Assets->Import Package->Custom Package… choose ‘3dGameKitWwise.unitypackage’, and import all assets

  3. Open Unity and then open the Start scene. Hit play in Unity to test project and hear audio from Wwise.

  4. You can remotely connect Wwise in realtime to profile the audio playing in the game

  5. Download the 3dGameKitWwise.zip. This contains the Wwise session and Unity package

  6. Download Unity Hub

  7. Install Unity 2022

  8. Create new Unity project called 3dGameKitWwise

  9. Download and install 3dGameKit from Unity Asset Store

  10. Close Unity

  11. Unzip the provided Wwise session into the new 3dGameKitWwise folder

  12. Download the Wwise Launcher

  13. Install Wwise 2023

  14. Copy 3dGameKitWwise Wwise session into the Unity project folder

  15. Integrate Wwise into 3dGameKit Unity project, make sure to reference the 3dGameKitWwise Wwise session

  16. Open the 3dGameKitWwise Unity project

  17. Run the game to test and profile in Wwise
